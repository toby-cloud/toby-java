## Install gradle

`curl -s https://get.sdkman.io | bash`

Open new terminal

`sdk install gradle 3.1`


## Usage

```bash
gradle tasks
gradle build
java -jar build/libs/toby-java-0.1.jar
```

## Dependencies

- [fusesource/mqtt-client](https://github.com/fusesource/mqtt-client)
- [google/gson](https://github.com/google/gson)
